Instrucciones

  • 1.- El objetivo de este curso es conocer a profundidad los elementos de la pirámide
  • 2.- Se inicia de manera descendente, comenzando con Carácter y Competencias
  • 3.- Al finalizar las lecturas de cada elemento hay una evaluación para retroalimentar lo aprendido.
  • 4.- Para continuar con el siguiente nivel tienes que pasar el anterior con la evaluación.
